Homepage (.cta / .s-btn / .lb-btn)

.cta: weight 800 · 15px · pad 14/28 · radius 12  ·  .s-btn: radius 9  ·  .lb-btn: radius 10
Audit: one visual idea (green primary + outlined ghost) implemented four times with drifting specs — radius 9/10/12, weight 700/800, font 13–16px, three different class vocabularies (btn / cta / s-btn / lb-btn). In the console nearly every action is the full-green primary, so hierarchy is flat. Proposal: one .btn with size (sm/md/lg) + intent (primary/ghost/danger/amber) variants, green reserved for the single most important action per view.
