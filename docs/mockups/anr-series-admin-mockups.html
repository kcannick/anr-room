No "locking" needed. The board only moves when a tagged session gets new votes — once the month's sessions are done, the top N are already final. Close series is just a status flip; the qualifiers are read live off the standings. (2.0: auto-seed an A&R Wars session with them.)

One operator flow: create a series (set its qualify-count) → tag each session inline from your session list (or add a new series on the spot) → watch the live leaderboard → lock the top N into A&R Wars at month's end.
